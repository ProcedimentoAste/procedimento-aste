Procedimento Aste — Guida alla pubblicazione del sito

Guida alla pubblicazione

Segui questi passi nell'ordine indicato. Tempo stimato: 20-30 minuti.

📁

Struttura dei file sul tuo computer

Prima di caricare su GitHub, assicurati che la cartella sul tuo computer sia organizzata esattamente così:

📁 procedimento-aste/ <-- questa è la cartella principale ├── procedimento-aste.html ├── procedimento-aste-lista.html ├── procedimento-aste-dettaglio.html ├── privacy-policy.html ├── cookie-policy.html ├── note-legali.html ├── netlify.toml ├── _redirects │ ├── 📁 admin/ │ ├── index.html │ └── config.yml │ ├── 📁 content/ │ ├── 📁 annunci/ │ │ ├── appartamento-via-roma-arezzo.json │ │ ├── locale-commerciale-arezzo.json │ │ └── terreno-san-giovanni-valdarno.json │ └── 📁 impostazioni/ │ └── studio.json │ └── 📁 uploads/ <-- lascia vuota per ora
1

Crea il repository su GitHub

GitHub è il "cassetto" online dove vengono salvati tutti i file del sito. Gratuito

Vai su github.com e crea un accountSe ce l'hai già, accedi direttamente.
Clicca "New repository" (tasto verde +)Nome: procedimento-aste — Visibilità: Public — poi "Create repository"
Carica i fileNella pagina del repository appena creato clicca "uploading an existing file", poi trascina tutto il contenuto della cartella procedimento-aste/ nel riquadro tratteggiato. Poi clicca "Commit changes".
Trascina il contenuto della cartella, non la cartella stessa. Apri la cartella procedimento-aste/ e seleziona tutti i file e sottocartelle dentro di essa.
2

Pubblica il sito su Netlify

Netlify legge i file da GitHub e li pubblica su internet automaticamente. Gratuito

Vai su netlify.com → "Sign up with GitHub"Autorizza l'accesso quando richiesto.
Clicca "Add new site" → "Import an existing project" → GitHubScegli il repository procedimento-aste
Impostazioni deploy: Branch: main — Build command: lascia vuoto — Publish directory: . (un punto) — poi "Deploy site"
Il sito sarà online in 1-2 minuti su un indirizzo tipo nome-casuale.netlify.app. Puoi rinominarlo in Netlify → "Site configuration" → "Site details" → "Change site name".
3

Attiva il pannello di gestione (Identity)

Questi passaggi ti permettono di accedere al pannello admin per gestire gli annunci.

Segui i passi nell'ordine esatto. Non saltare nessuno.
1. Netlify → "Site configuration" → "Identity" → "Enable Identity"
2. Nella stessa pagina, sezione "Registration" → seleziona "Open"Questo ti permette di registrarti direttamente dal sito senza aspettare email di invito.
3. Sempre in Identity → sezione "Services" → "Git Gateway" → "Enable Git Gateway"
4. Vai su: https://tuosito.netlify.app/adminSi aprirà il pannello con un bottone "Login with Netlify Identity". Cliccalo → poi clicca "Sign up" → inserisci la tua email e una password → conferma.
5. Dopo esserti registrato, torna su Netlify → Identity → "Registration" → rimetti "Invite only"Così nessun altro potrà registrarsi.
Sei dentro il pannello admin! Da questo momento puoi aggiungere, modificare e cancellare annunci senza toccare il codice.
4

Come aggiungere un annuncio

Dal pannello admin ogni volta che vuoi pubblicare un nuovo immobile.

Vai su tuosito.netlify.app/admin → accedi con email e password
Clicca "Annunci Immobili" → "New Annuncio"
Compila i campi: tipologia, sottotipologia, titolo, indirizzo, città, provincia, offerta minima, descrizione, foto copertina, galleria
Clicca "Save" → poi "Publish"L'annuncio appare sul sito in 30-60 secondi.
Dopo aver creato un nuovo annuncio, devi aggiungere il nome del file anche in procedimento-aste-lista.html nella riga const ANNUNCI_FILES = [...]. Il nome del file viene creato automaticamente dal titolo (tutto minuscolo, spazi sostituiti da trattini). Es: se il titolo è "Appartamento Firenze", il file sarà appartamento-firenze.json.
